# Resume

This is my [resume](resume.pdf) generated using [LaTeX](https://www.latex-project.org/).

## How it Works

Install [Docker](https://www.docker.com/), modify [resume.tex](resume.tex), and run

    make

to re-generate [resume.pdf](resume.pdf).
